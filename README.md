# DevOps Automation Agent
AI Agent for DevOps Automation - AssembleHack25

**Status:** Work in Progress (AssembleHack25)

An intelligent AI agent that automates DevOps workflows using modern orchestration tools.

# Project Goals

Build an AI agent that:
- Analyzes code changes using Cline CLI
- Orchestrates deployment workflows with Kestra
- Makes intelligent decisions using Oumi
- Provides real-time status via Vercel dashboard

# Tech Stack:
- **Cline CLI**: Code analysis and script generation
- **Kestra**: Workflow orchestration
- **Oumi**: AI agent reasoning with open-source LLMs
- **Vercel**: Dashboard deployment
- **CodeRabbit**: Code quality reviews
